Tutorial

O que é o jQuery Countdown Timer ?

R: O jQuery Countdown é um plugin adaptado para ser usado em qualquer layout, sem qualquer dependência CSS / HTML. O objetivo é ajustar e imitar diferentes estilos de contagem regressiva como você vê em cupons e sites de leilão. Podendo utilizar um áudio para quando a contagem acabar.
Existem algumas maneiras de começar, o exemplo mais simples é o mostrado aqui.

Como utilizar o plugin ?

Para começar a utilização do plugin você deve acessar o seguinte link, e fazer o download dos arquivos, usaremos apenas a pasta dist.

jQuery Countdown

Em seguida, crie um arquivo HTML, feito isso você deve criar um script chamando o timer.jquery.min.js

<script src="js/jquery-3.3.1.js"></script>
<script src="js/timer.jquery.min.js"></script>

Para começar a escrever o código do cronômetro você deve criar um id chamado Tempo que vai servir para o relógio, como na imagem abaixo

  <div class="container-fluid">
      <div class="text-center">
        <br/>
        <br>
        <h2 class="text-warning">Temporizador</h2>
        <div class="jumbotron col-md-4 col-md-offset-4">
          <h1 id="Tempo">00:00:00</h1>
    </div>


Feito isso, é hora de criarmos as divs que serão usadas para o relógio, criaremos a div para as horas, os minutos e os segundos, e criaremos um botão para iniciar o cronômetro. Como na imagem abaixo

<div class="row">
          <div class="col-md-4 col-md-offset-4">
          <div class="col-md-4">
            Horas
            <input class="form-control" type="number" id="h" value="0" min="0" max="23">
          </div>
          <div class="col-md-4">

            Segundos

          <br/>

        </div>
          </div>

        </div>

      </div>

    </div>


Chegou o momento de colocarmos um aviso sonoro para quando o tempo acabar e colocarmos o cronômetro para funcionar. Você deve criar uma pasta e colocar o audio que você deseja utilizar em seu site nessa pasta. Em seguida crie uma função e coloque o código necessário para que o plugin funcione. Após essas etapas feitas, criaremos a duração para que o temporizador funcione.

<script>
    audio = new Audio("som/alarme.mp3");
      function comecar(){
          audio.pause();
          audio.currentTime=0;
          $("#Tempo").timer('remove');
          $("#Tempo").timer({
              countdown:true,
              duration:$('#h').val()+"h"+$('#m').val()+"m"+$('#s').val()+"s",
              callback:function(){
                //Tempo se terminar
                audio.addEventListener('ended',function(){
                  this.currentTime=0; // Loop
                  this.play();
                },false);
                audio.play();


              },
              format:'%H:%M:%S'
          });

      }

    </script>
